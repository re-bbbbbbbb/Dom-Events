Dom Events


Reazione agli eventi

Javascript può essere eseguito quando un evento viene verificato, un esempio è quando l'utente clicca su un elemento HTML.

Per eseguire il codice quando l'utente fa clic sull'elemento, bisogna aggiungere al codice JavaScript un attributo, cioè onclick = JavaScript.

Alcuni esempi degli eventi sono:

  • Un utente fa clic con il mouse;
  • Una pagina web è stata caricata;
  • Un'immagine è stata caricata;
  • Il mouse viene spostanto su un elemento;
  • Un campo input viene modificato;
  • Un form HTML viene presentato;
  • Un utente preme un tasto.

Gli eventi onmouseover e onmouseout

L'onmouseover e l'onmouseout possono essere utilizzati per attivare una funzione quando l'utente passa con il mouse sopra o fuori da un elemento HTML.

Gli eventi onmousedown, onmouseup e onclick

L'onmousedown, l'onmouseup e l'onclick sono sono tutti eventi del clic del mouse.

  • Onmousedown --> attivato quando viene premuto il pulsante del mouse;
  • Onmouseup --> attivato quando il pulsante del mouse viene rilasciato;
  • Onclick --> attivato quando il clic del mouse è completato.

Altri dom events

Sono presenti anche altri eventi, ovvero:

  • Onfocus --> cambia il colore di sfondo di un campo di input quando viene messo in evidenza;
  • Mouse events --> cambia il colore di un elemento quando il cursore si sposta su di esso.
